# reeshabh.in

[![Netlify Status](https://api.netlify.com/api/v1/badges/aebe8ab7-8862-4400-979d-dc572988d939/deploy-status)](https://app.netlify.com/projects/magenta-sawine-f25b80/deploys)

Personal portfolio website built with React and TypeScript, hosted on Netlify.

## Features

- Responsive design with Tailwind CSS
- Experience showcase with company logos and tags
- Project highlights
- Contact form with Netlify Forms integration
- Optimized for all devices

## Development

```bash
npm install
npm start
```

## Deployment

The site automatically deploys to Netlify on push to the `main` branch.
